# Groundwater Mapping Mobile App

A mobile application for groundwater mapping built with Google Stitch.

## Screens

| Screen | Description |
|--------|-------------|
| `ar_subsurface_view` | AR visualization of subsurface layers |
| `geological_analysis` | Geological data analysis |
| `geological_analysis_updated_export` | Updated export view for geological analysis |
| `gis_metadata_view` | GIS metadata display |
| `real_time_ground_scan` | Real-time ground scanning interface |
| `sensor_connectivity` | Sensor connection management |

Each screen folder contains:
- `code.html` — screen source code
- `screen.png` — screen preview
